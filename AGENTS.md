This repository contains the official public website of **GlebMar Apps**.

It is **NOT** the Guardian Android application.

It is the publisher website.

The repository is deployed through **GitHub Pages**.

# Technology

Static website only.

Allowed:

- HTML
- CSS
- Minimal Vanilla JavaScript

Hosting:

GitHub Pages

No backend.

No server.

No build pipeline.

---

# Content Rules

The website must never:

- teach scammers;
- describe detection methods;
- expose algorithms;
- promise perfect protection;
- promise impossible behavior;
- invent future products;
- publish fake testimonials;
- publish fake statistics;
- publish fake download counts.

Only describe real products.

# Performance

Keep the website lightweight.

Do not introduce:

- React
- Vue
- Angular
- Next.js
- Astro
- build tools
- package managers
- analytics
- tracking scripts
- cookies
- third-party JavaScript

# Accessibility

Always maintain:

- semantic HTML;
- keyboard navigation;
- visible focus states;
- high contrast;
- responsive layouts;
- reduced-motion support.

Accessibility is not optional.

# Repository Rules

Never:

- store secrets;
- store passwords;
- store Google credentials;
- store keystores;
- store identity documents;
- commit release artifacts.

This repository is public.

# Scope Rules
